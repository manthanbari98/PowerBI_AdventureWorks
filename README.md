# PowerBI_AdventureWorks
End-to-end Power BI project involving data import, transformation, and sales analysis, with interactive dashboards for business insights and employee details dashboard

Welcome to the **PowerBI_AdventureWorks** repository!

This project demonstrates a comprehensive data import and transformation using DAX and altering M in PowerQuery, for building interactive dashboard to derive actionable insights.
``` ```
## Project Requirements

### Objectives
Create a dashboard in PowerBI using data to build interactive dashboard to derive business insights.

### Specifications
- **Data Sources**: Import data provided as csv.
- **Data Quality**: Cleanse and resolve data quality issues prior to analysis.
- **Integration**: Creating user friendly data model using DAX and altering M in PowerQuery.
- **Scope**: Interactive dashboard to derive business in-sight.
- **Documentation**: Provide clear documentation of the data model to support both business and analytics teams.
``` ```

### Objectives
Develop Dashboard to  deliver insights into:
- **Sales trend**
- **Top selling products**
- **Employee Details**

These insights empower stakeholders with key business metrics, enabling strategic decision-making.
